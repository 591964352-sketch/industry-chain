# 豆包查询标准模板

每次需要让豆包查数据时直接套用，无需等顾问生成提示词。

## 硬约束（每次必须包含，不得删减）

- 只使用巨潮资讯 / 东方财富 / 同花顺 三个数据源
- 查不到的字段填"暂无"，不得推断或编造数据
- 每条数据必须注明来源网站和数据时间
- 客户名称只引用券商研报，不自行推断，标注 broker🔵
- 所有精确数字必须附原始页面链接，无链接标注"待核实"
- 查询完成后结果原文转发给顾问，不做任何修改或补充
- 数据来源分级：primary🟢（巨潮原文）> broker🔵（券商研报）> media⚪（东财/同花顺）

## 防造假规则

- 精确数字（百分比/金额/排名）必须附巨潮/东财/同花顺原始链接
- 客户名称来自券商研报时标注 broker🔵，不写入 pcb.manual.js，只作参考
- 豆包返回精确数字后，CC 用巨潮链接核对原始出处，核对通过才写入
- barrier / dims6 / reduceSignal 三类字段豆包只提供素材，顾问判断后 CC 写入

## 标准模板格式

```
查询日期：[YYYY-MM-DD]
查询目的：[一句话说明用途]
硬约束：只用巨潮/东方财富/同花顺，查不到填暂无，每条注明来源和时间。
查询内容：[具体查询项]
输出格式：字段对齐，来源和时间逐项注明，精确数字附链接。
查完结果原文转发给顾问。
```

## 常用查询类型

### 壁垒核实类
1. 是否通过[客户名]核心供应商认证（是/否+链接）
2. 核心技术壁垒（一句话，年报原文）
3. 竞争对手是否有同等能力（是/否）
4. 市场份额/排名（数字+链接）

### 基本面财务类
1. 最新收盘价（元，截至日期）
2. PE(TTM)（倍，来源）
3. PE 历史分位（近 3 年，来源）
4. 最新毛利率（%，最近一期财报+链接）
5. 最新 ROE 加权（%，来源+链接）
6. 近 1 个月涨跌幅（%）

### 景气数据类
1. [原材料]最新价格（元/吨+链接）
2. 近 3 个月趋势（上涨/下跌/平稳）
3. 行业景气度最新判断（一句话+来源）

### 持仓监控类
1. 最近 5 日收盘价（逐日，来源）
2. 主力资金近 5 日净流入/流出（万元，来源）
3. 近期大股东减持公告（有/无，若有列出+链接）
4. 分析师一致预期（来源注明）
