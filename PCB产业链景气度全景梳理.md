# PCB 产业链景气度全景梳理（截至 2026 年 6 月）

> 本文为产业与个股研究梳理，所有数据来自公开资料（Prismark、各券商研报、上市公司公告、行业协会等），市占率/份额多为不同口径下的近似值，仅供研究参考，**不构成任何投资建议**。涉及估值与买卖判断请独立决策或咨询持牌投顾。

---

## 一、宏观层：本轮处于「AI 驱动的结构性上行周期」

### 1. 周期定位
PCB 是典型的强周期成长行业。2022–2023 年受消费电子去库存拖累下行（2023 年全球产值约 783 亿美元，同比 -4.2%），**2024 年触底回升**（+5.8%），2025 年起在 AI 算力拉动下进入新一轮上行周期。

| 维度 | 全球 | 中国大陆 |
|---|---|---|
| 2024 年产值 | 约 735–880 亿美元（口径不一，+5.8%） | 412 亿美元 / 约 4121 亿元，占全球 **56%** |
| 2025 年（E） | 786–968 亿美元 | 437 亿美元 / 约 4333 亿元（+6.1%） |
| 2029 年（E） | 约 940–946 亿美元 | 约 497 亿美元 |
| 长期 CAGR（25–29） | 约 4.8–5.2% | 约 3.3–4.3% |

A 股 PCB 板块 2025H1 营收约 **1327 亿元（+25.2%）**、归母净利约 **124 亿元**，景气向上趋势明确。

### 2. 核心驱动力（按重要性排序）
1. **AI 服务器（绝对主线）**：单机 PCB 价值量较传统服务器提升 **5–10 倍**（UBS 测算），远超 4G→5G 时的 2–3 倍。英伟达 GB200/300 放量、2026 年 Rubin 平台、2026–27 年中板/背板新设计将持续抬升单机价值。2024–2029 全球服务器市场 CAGR 约 18.8%，加速服务器支出 CAGR 20%+。
2. **国产算力供应链**：华为、寒武纪、摩尔线程等自主芯片集群放量，带动本土高端 PCB/CCL 国产替代。
3. **汽车电子**：单车 PCB 用量从传统车 0.6–1㎡ 提升至新能源 5–8㎡，ADAS 带动高频雷达板/HDI。
4. **消费电子温和复苏 + AI 端侧**（AI 手机/AIPC）带动 SLP、高阶 HDI。

### 3. 一句话景气判断
**总量温和复苏 + 高端结构性爆发**。普通板（单双面、低端多层）增速接近 GDP；而 18 层以上高多层板、IC 载板、高速 CCL、HVLP 铜箔等高端环节量价齐升，是本轮景气的「真核心」。Prismark 预测 2024–2029 全球 18 层以上多层板 / IC 载板 / HDI 的 CAGR 分别为 **15.7% / 7.4% / 6.4%**（中国大陆 18 层以上达 21.1%），显著高于行业 5% 的均速。

## 二、行业层：产业链结构与价值分布

### 1. 三段式全景
```
上游：原材料 + 专用设备  →  中游：PCB 制造（含载板） →  下游：终端应用
  │                              │                          │
  ├ 覆铜板 CCL（核心）           ├ IC 载板/封装基板          ├ 服务器/数据中心（增量最强）
  ├ 树脂 / 电子布 / 铜箔         ├ 高多层板（服务器/背板）   ├ 手机 + PC（存量最大）
  ├ 油墨 / 干膜 / 半固化片       ├ HDI 板                    ├ 汽车电子
  └ 钻孔/曝光/电镀/检测设备+钻针 ├ FPC 软板                  ├ 通信/网络设备
                                └ 普通刚性板（单双面/多层）  └ 工业/医疗/航空国防
```

### 2. 成本结构（PCB 制造视角）
- **直接材料约占成本 50%**，其中 **覆铜板（CCL）占 ~27.3%** —— 这是 CCL 厂商对 PCB 厂利润有强传导能力的根源。
- CCL 自身成本拆分：**铜箔 + 树脂 + 玻璃纤维布** 三大件，铜箔通常占比最高。
- 这意味着「**铜价 / 树脂 / 玻纤布涨价 → CCL 涨价 → PCB 厂成本上行**」的传导链条；本轮高端材料缺货背景下，上游材料环节弹性甚至大于中游制造。

### 3. 中游产品结构占比（2024，Prismark）

| 产品类别 | 全球占比 | 中国大陆占比 | 备注 |
|---|---|---|---|
| 多层板（含高多层） | **38%** | 44.9% | 占比最高，高多层是 AI 主战场 |
| IC 载板/封装基板 | 17% | 7.5% | 中国大陆占比远低于全球，国产替代空间最大 |
| FPC 软板 | 17% | 14.5% | 消费为主，向汽车拓展 |
| HDI 板 | 17% | 19.0% | AI 服务器 + 高端消费 |
| 单/双面板 | 11% | 14.0% | 壁垒最低、增速最慢 |

## 三、壁垒梯队总览（全产业链分支排序）

> 「壁垒」综合考量：技术认证周期、良率难度、客户绑定、资本开支、上游卡脖子程度。

| 梯队 | 分支环节 | 壁垒来源 | 国产化现状 |
|---|---|---|---|
| **T0 最高** | ABF 载板 | 良率/线宽（<8μm）、味之素 ABF 膜垄断、3–5 年客户认证 | 内资近乎空白，刚起步 |
| **T0 最高** | 高速 CCL（M8/M9/M10） | 材料配方 + 树脂/铜箔/电子布协同、终端认证 | M8 量产、M9 突破中 |
| **T1 高** | 高端电子布（石英布/Low-DK 二三代布） | 全球仅日东纺/圣戈班/菲利华等少数、产能极紧 | 中材科技/菲利华突破 |
| **T1 高** | 高多层板（≥18~28 层服务器/背板） | 层数对位/孔加工/良率、超高速材料适配 | 头部厂已领先全球 |
| **T1 高** | HVLP 铜箔（HVLP4/5+） | 表面处理工艺、信号损耗控制 | 铜冠/德福突破中 |
| **T2 中高** | BT 载板 | 客户认证、量产经验 | 深南/兴森领先 |
| **T2 中高** | HDI（5 阶及以上 / Anylayer） | 激光钻孔/镀铜/对位 | 头部成熟 |
| **T2 中高** | 高速树脂（PPO/碳氢/BMI/活性酯） | 配方与一致性、覆铜板厂认证 | 东材/圣泉突破 |
| **T2 中高** | 高端 PCB 设备（曝光 LDI/钻孔/电镀） | 精度与稳定性 | 国产化率 <30% |
| **T3 中** | 常规多层板 / 汽车板 | 车规认证、规模与成本 | 成熟 |
| **T3 中** | FPC 软板 | 良率与客户绑定（苹果链） | 鹏鼎/东山领先 |
| **T3 中** | 钻针等耗材 | 精密磨削、涂层 | 鼎泰全球第一 |
| **T4 较低** | 单/双面板、普通刚性板 | 规模成本为主 | 充分竞争、内卷 |

### 【中游 · T0】IC 载板 / 封装基板 —— 壁垒最高、国产替代空间最大

**规模与逻辑**：占全球 PCB 17%，但中国大陆内资仅占全球 **3.2%**（2022，BT 7%、ABF 几乎空白）。全球被中国台湾（欣兴）、日本（Ibiden 揖斐电）、韩国（三星电机）垄断，前十占 80%+。ABF 载板上游 ABF 膜由日本味之素独家垄断。这是 PCB 产业链「金字塔尖」，也是国产替代弹性最大的方向。

### 【上游 · T0】覆铜板 CCL（高速/高频） —— 材料卡位主线

**规模与逻辑**：CCL 占 PCB 成本 ~27%，是上游最关键环节。高端 CCL（高速 M 系列、高频）认证壁垒高、当前阶段性缺货。AI 服务器 CCL 用量是传统设备 **3–5 倍**，M9/M10 迭代加速，2024–2027 高端 CCL 市场 CAGR 约 **40%**。

**全球刚性 CCL 份额（2023，CR5≈54.8%）**：建滔 14.7%、生益科技 14.0%、台光电材 10.3%、南亚塑胶 9.2%、松下 6.6%。
**高频 CCL**：罗杰斯、AGC、三菱瓦斯、依索拉、松下主导，CR5>80%。

## 五、横向对比：谁在进步 / 谁相对掉队

### 明确「进步」（卡位 AI 高端、份额/利润双升）
- **中游硬板**：胜宏科技、沪电股份、生益电子、深南电路、世运电路
- **上游材料**：生益科技、南亚新材、华正新材（CCL）；东材科技（树脂）；宏和科技、中材科技（电子布）；铜冠铜箔、德福科技（铜箔）
- **设备耗材**：鼎泰高科、大族数控
- **载板国产替代**：深南电路、兴森科技（长期，短期受折旧压制）

### 相对「掉队 / 需警惕」（非贬义，指本轮弹性弱）
- **传统消费导向、AI 暴露低**的厂商：纯消费 FPC、低端单双面/普通多层板厂，享受不到高端结构性红利。
- **龙头守擂点**：鹏鼎、东山精密体量全球领先，但收入仍以消费电子（苹果链占比高）为主，AI 服务器暴露相对低，处于「不加大 AI 投入就可能被甩开」的竞争守擂阶段——正加速转型。
- **扩产阵痛**：扩产过猛的厂商短期面临折旧、用工成本、NPI 导入对利润的压制（如个别龙头单季环比曾下滑），是节奏问题而非趋势反转。

## 六、风险提示
1. **AI 资本开支不及预期**：海外云厂商 Capex 是核心变量，若放缓则高端环节景气回落。
2. **高客户集中度**：头部厂商绑定单一大客户（如英伟达/苹果），客户产品迭代/砍单风险大。
3. **扩产过剩**：行业集体「军备赛」扩产，2025–26 头部投资额数百亿，未来 2–3 年高端产能集中释放，可能引发价格战。
4. **上游卡脖子**：ABF 膜（味之素）、高端电子布（日东纺等）供给瓶颈，制约国产载板/M9 放量节奏。
5. **汇率波动**：出口占比高的厂商（胜宏出口近八成）汇兑损益对利润影响加大。
6. **估值已较高**：部分龙头 2023 年底至今涨幅数倍，PE 处历史高位，需注意业绩兑现与估值匹配。
