# 产业链全景 · 每日使用指南

## 🎯 目标：每日点击收藏夹即看到完整网页

## 📋 一次性配置（5 分钟搞定）

### 步骤 1：双击 `install-startup.bat`
把 `start-silent.vbs` 的快捷方式复制到 Windows 启动文件夹，实现**开机自启**。

### 步骤 2：浏览器添加收藏夹
- 地址：`http://localhost:8765/index.html#pcb`
- 名称：建议「产业链全景」

### 步骤 3：完成
- 每日开机后 5-10 秒，两个 server 自动启动
- 浏览器点收藏夹即可看到网页（含机构信号）

---

## 🔧 手动启动 / 调试时

如果开机自启出问题，可手动启动：

**方法 A：双击 `start-silent.bat`**
- 会弹出黑色 cmd 窗口（调试用，看得到启动日志）
- 启动完成后窗口自动关闭（server 仍在后台跑）

**方法 B：双击 `start-silent.vbs`**
- 不弹任何窗口，静默启动
- 与开机自启效果一致

---

## 📂 端口说明

| 端口 | 作用 | 何时启动 |
|---|---|---|
| **8765** | 静态文件 server（HTML/CSS/JS/data）| 每日启动 |
| **8081** | 机构信号 server（北向资金 + 基金持仓）| 每日启动 |

如果 8081 没起 → 页面仍可打开，但 stock 卡片的"机构信号行"会显示"数据获取失败"

---

## 🚨 故障排查

| 现象 | 排查步骤 |
|---|---|
| 网页打不开（无法连接 8765）| 双击 `start-silent.bat` 看启动日志 |
| 网页打开了但机构信号显示"数据获取失败" | 检查 8081 端口是否被占用（`netstat -ano \| findstr ":8081"`）|
| 两个 server 都启动了但网页打不开 | 浏览器按 Ctrl+Shift+R 硬刷一次 |
| 端口被占用 | 杀掉占用进程：`taskkill /F /PID <pid>` |

---

## 🛠️ 卸载开机自启

1. 按 `Win + R` → 输入 `shell:startup` → 回车
2. 删除 `产业链全景-启动器.lnk`
3. 完
